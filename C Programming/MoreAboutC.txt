Simple: A program runs through compilation, loading, and execution.
First, the compiler converts source code into machine code.
Then the OS loader loads it into RAM and sets up stack/heap.
The CPU then executes the machine instructions using the Fetch-Decode-Execute cycle.
During execution, the program interacts with memory and OS through system calls.
Finally, when execution ends, the OS frees all resources.
